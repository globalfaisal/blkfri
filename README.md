# BLKFRI

A simple e-commerce website built with:

- Next.js
- Sanity
- Clerk
- Stripe
- Shadcn
- Tailwind CSS

## Getting Started

First, run the development server:

```bash
npm run dev
```

### Development URLs

Website: [http://localhost:3000](http://localhost:3000)

Sanity Studio: [http://localhost:3333/admin](http://localhost:3333/admin)
